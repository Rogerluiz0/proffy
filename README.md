## Installation

- `git clone https://github.com/Rogerluiz0/proffy.git`;
- `cd proffy`;
- `npm install `;
- `npm run dev`.

Requirements to run **locally**:

- [ ] [Node.js](https://nodejs.org/en/) ( Required )
- [ ] [GIT](https://git-scm.com/) ( Optional )
